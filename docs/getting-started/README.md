Use this page if you want Sentrio working in a few minutes.

## 5 minute setup

1. Install and open Sentrio.
2. Click the menu bar icon.
3. Open **Preferences...**.
4. In **Output** and **Input** tabs, drag devices into your preferred order.
5. Keep **Auto** enabled.

That is enough for automatic switching.

## What to check next

- Output switches when headphones are connected/disconnected.
- Input switches when your preferred mic appears.
- Volume memory restores expected levels.
