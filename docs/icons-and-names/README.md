You can customize device identity without changing hardware behavior.

## Custom icons

- Right-click a device in the menu panel.
- Choose a different icon.
- Icon choice is stored per role (Output vs Input).

## Custom names

- Rename devices in Preferences.
- Names are also role-specific.
- Useful when the same physical device appears in multiple profiles.

## Battery and warnings

- Battery icons appear when macOS provides battery data.
- Very low battery on active devices shows a red warning icon in the menu bar.

## Tips

- Keep names short so menu rows stay readable.
- If you use AirPods, expect separate profiles for output and headset mode.
