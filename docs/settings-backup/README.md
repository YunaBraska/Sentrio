Use this page for app-wide options and safe migration.

## General settings

- Auto-switching master toggle
- Launch at login
- Hide menu bar icon
- Input level meter visibility in Preferences
- Language selection

## Export settings

Use **Export Settings...** to create a JSON backup.

## Import settings

Use **Import Settings...** to restore from a previous export.

## What backup includes

- Output/Input priority lists
- Disabled devices
- Custom names and icons
- Volume memory
- BusyLight settings and rules

## Good practice

- Export before major changes.
- Keep at least one known-good backup file.
